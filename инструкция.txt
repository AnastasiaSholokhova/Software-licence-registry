Запуск приложения:
waitress-serve --host 127.0.0.1 --port 5000 app:app
ссылка: http://127.0.0.1:5000
Приложения: 
1. app_online.py - подключение к облачной базе данных через поле DATABASE_URL. Хост- 0.0.0.0
2. app_local.py - подключение к локальной базе данных postgresql. Переменные окружения в начале кода. Хост - 127.0.0.1
3. Пароль от Pgadmin4 - 12345
Сервер: db_original
Users - база данных пользователей (авторизация)
software_licences - все данные по лицензированию ПО
4. 16 - папка с postgresql. Там лежит pgadmin и базы данных.

Описание перенесенных файлов:
1. templates - папка со стилями html
2. static/css - стиль css
3. venv - виртуальное окружение с библиотеками
4. dockerfile - файл с инструкциями для контейнеризации приложения
